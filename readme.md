# Simulador 2D de Fluidos (Navier-Stokes)

![Simulación en acción](https://i.imgur.com/3jK8vL2.gif)

Simulación en tiempo real de **flujo laminar alrededor de un obstáculo circular** usando ecuaciones de Navier-Stokes 2D.

## Tecnologías
- Python
- NumPy
- Matplotlib

## Cómo ejecutar
```bash
pip install numpy matplotlib
python main.py
